# Daslash
2D action game made with PICO-8

# スプライトのフラグ
| フラグ    | 意味           |
| --------- | -------------- |
| 0000 0001 | 右突き抜け禁止 |
| 0000 0010 | 左突き抜け禁止 |
| 0000 0100 | 下突き抜け禁止 |
| 0000 1000 | 上突き抜け禁止 |
| 0001 0001 | ゴール         |
| 0001 0100 | はしご         |
